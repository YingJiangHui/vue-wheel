# 快速上手

## 资源

```js
import {Button} from 'eagle-ui'
import "eagle-ui/dist/index.css";
```

## 注册组件

```js
export default {
    components:{
        'e-button':Button
    }
}
```

## 使用

```html
<template>
    <div>
        <e-button></e-button>
    </div>
</template>
```
